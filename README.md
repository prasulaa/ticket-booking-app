# Ticket booking app

Java + Spring Boot + H2 database

## Running application

``` mvn spring-boot:run ```
or
``` ./run-app.sh ```

## Running use case scenario

``` ./run-use-case.sh ```

## Helpful endpoints

### Swagger UI

``` /swagger-ui/ ```
